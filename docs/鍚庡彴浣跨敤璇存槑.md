# MAGO 网站后台使用说明

## 登录后台

访问 `/studio`，输入管理员密码。

上线前请在环境变量中配置：

- `CMS_ADMIN_PASSWORD`
- `SESSION_SECRET`

## 更新活动

进入“活动管理”：

1. 点击“新增活动”或选择已有活动。
2. 填写活动名称、日期、城市、场地和时间。
3. 上传带小程序码的活动海报。
4. 粘贴小程序官方 URL Link。
5. 设置活动状态。
6. 选择是否首页推荐、是否前台显示。
7. 点击“保存内容”。

### 活动状态

- 即将开售：前台展示开售状态，不跳转购票。
- 立即购票：手机点击后打开小程序 URL。
- 余票紧张：手机点击后打开小程序 URL，并显示紧张状态。
- 已售罄：禁止购票跳转。
- 活动取消：禁止购票跳转。
- 活动结束：归入历史活动。

## 更新城市

进入“城市管理”，可以：

- 新增或删除城市
- 修改城市名称和 URL 标识
- 上传城市封面
- 修改城市活动说明
- 设置是否前台显示

## 更新图库

进入“图库管理”：

- 上传活动照片
- 修改图片标题、活动、城市、年份和说明
- 隐藏或删除图片

建议横版图片宽度不低于 2000px。

## 更新活动须知

进入“活动须知”，可以新增、删除、修改问题和回答。

## 更换微信二维码

进入“微信联系”：

- 上传客服微信二维码
- 修改微信号
- 修改联系人名称、服务时间和城市

客服微信二维码与活动海报中的小程序码为两套独立素材，不要混用。

## 修改页面文字

进入“页面文案”，可以修改导航、首页、活动页、图库页、城市页、购票弹窗、页脚、隐私政策和 404 页面中的全部文案。

固定内容：

- `MAGO`
- `SUNSET`

其他前台可见文案均已放入后台。

## 首屏视频

进入“网站设置”，上传 MP4 或 WebM 文件并保存。未上传视频时，首页会自动轮播活动现场照片。
